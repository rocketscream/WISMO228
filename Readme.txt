Readme
******
1. This is an Arduino compatible library for Sierra Wireless's WISMO228 GSM-GPRS module which is used on our TraLog shield.
2. Latest version is v1.20:
- Added support for hardware serial (on Arduino Mega2560 & Leonardo).
- Software serial object needs to be declared outside of the library.
- Added support for ring indicator (new SMS & incoming call indication) for Arduino Leonardo.
- Supports Arduino IDE 1.6.5
3. All examples are updated to v1.10.
